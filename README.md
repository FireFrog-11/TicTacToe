# Multiplayer TicTacToe

This is a small project I created using **Unity Netcode for GameObjects** to create simple multiplayer that supports Client/Host and fully synchronized game state.

## Features
- Fully functional TicTacToe game with win-checking and turns
- Host/Client multiplayer using Unity NGO
- Synced game board and player turns
- Unique UI for each player
- Background music and sound effects
- Rematch button and tracks score


## How to Play
1. Download the **1.0 Release** from GitHub.
2. Unzip the folder.
3. Duplicate the folder (one copy for Host, one for Client).
4. Run `TicTacToe.exe` in each folder.
5. On one copy, click **Host**.  
   On the other, click **Client**.
6. Enjoy playing TicTacToe!
